Sala de Seguranca VR

Projeto Final - Unidade 1, Capitulo 3 - Web 3.0 / Residencia em TIC 29.

Repositorio:
https://github.com/duckrwx/cftv-metaverso

Como abrir:
1. Clone o repositorio.
2. Abra a pasta do projeto no Unity Hub.
3. Use a versao Unity 6000.3.19f1 ou uma versao compativel.
4. Abra a cena Assets/Scenes/SalaSegurancaVR.unity.
5. Pressione Play no Unity Editor.

Controles no PC:
- W, A, S, D: mover.
- Mouse: olhar ao redor.
- Espaco: subir.
- Ctrl: descer.
- Shift: mover mais rapido.
- Esc: liberar ou travar o cursor.

Observacao:
O projeto foi preparado para teste local no Unity Editor. A configuracao completa para Meta Quest deve ser validada no Unity Editor com XR Plugin Management, Android Build Support e Meta XR SDK, se exigida na correcao.
